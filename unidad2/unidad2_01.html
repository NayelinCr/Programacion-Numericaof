1. M√©todo de Newton-Raphson

Definici√≥n

Es un procedimiento iterativo que utiliza la **derivada de la funci√≥n** para aproximar las ra√≠ces de ecuaciones no lineales. Su rapidez de convergencia (convergencia cuadr√°tica) lo hace muy utilizado en ingenier√≠a y matem√°ticas aplicadas.

F√≥rmula

$$x_{n+1}=x_{n}-\frac{f(x_{n})}{f^{\prime}(x_{n})}$$

Acotaciones

  • Convergencia r√°pida si la aproximaci√≥n inicial est√° cerca de la ra√≠z verdadera.
  • Requiere que la funci√≥n sea derivable en el intervalo considerado.
  • Puede fallar o divergir si la derivada en el punto inicial es cero o cercana a cero.

1.1. C√≥digo en Python



import math

# Funci√≥n a resolver: f(x) = e^x - 3x
def f(x):
    return math.exp(x) - 3*x

# Derivada: f‚Äô(x) = e^x - 3
def df(x):
    return math.exp(x) - 3

def newton_raphson(x0, tol=1e-6, max_iter=100):
    """
    Aproxima la ra√≠z de f(x) usando el m√©todo de Newton-Raphson.
    
    x0: valor inicial
    tol: tolerancia para convergencia
    max_iter: n√∫mero m√°ximo de iteraciones
    """
    xn = x0
    for n in range(max_iter):
        f_val = f(xn)
        df_val = df(xn)
        
        if abs(df_val) < 1e-12:
            print("Derivada cercana a cero. No se puede continuar.")
            return None
            
        xn1 = xn - f_val/df_val
        
        print(f"Iteraci√≥n {n+1}: x = {xn1:.8f}, f(x) = {f(xn1):.8f}")
        
        if abs(xn1 - xn) < tol:
            print(f"\nLa ra√≠z aproximada es: {xn1:.8f}")
            return xn1
            
        xn = xn1
        
    print("No se alcanz√≥ la tolerancia requerida.")
    return xn

2. M√©todo de Bisecci√≥n

Definici√≥n

Procedimiento iterativo que localiza una ra√≠z de una **funci√≥n continua** dividiendo repetidamente un intervalo en mitades. Aprovecha el **Teorema del Valor Intermedio**, garantizando convergencia cuando $f(a) \cdot f(b) < 0$.

F√≥rmula

$$x_{n} = \frac{a_n + b_n}{2}$$

Acotaciones

  • Siempre converge si la funci√≥n es continua y hay un cambio de signo en los extremos del intervalo.
  • No requiere c√°lculo de derivadas.
  • La convergencia es lineal, por lo que es m√°s lento comparado con m√©todos basados en derivadas.

2.1. C√≥digo en Python

import math

# Funci√≥n a resolver: f(x) = cos(x) - x
def f(x):
    return math.cos(x) - x

def biseccion(a, b, tol=1e-6, max_iter=100):
    """
    Encuentra la ra√≠z de f(x) en el intervalo [a, b] usando el
    m√©todo de Bisecci√≥n.
    
    a, b: extremos del intervalo
    tol: tolerancia para convergencia
    max_iter: n√∫mero m√°ximo de iteraciones
    """
    if f(a) * f(b) >= 0:
        print("El m√©todo no es aplicable: f(a) y f(b) deben tener signos opuestos.")
        return None
        
    for n in range(1, max_iter+1):
        m = (a + b)/2
        fm = f(m)
        
        print(f"Iteraci√≥n {n}: a={a:.6f}, b={b:.6f}, m={m:.6f}, f(m)={fm:.6f}")
        
        if abs(fm) < tol or (b - a)/2 < tol:
            print(f"\nLa ra√≠z aproximada es: {m:.6f}")
            return m
            
        if f(a)*fm < 0:
            b = m
        else:
            a = m
            
    print("No se alcanz√≥ la tolerancia requerida.")
    return m

a, b = 0, 1

3. M√©todo de la Secante

Definici√≥n

Aproxima la ra√≠z de una funci√≥n usando la **l√≠nea secante** entre dos puntos consecutivos. Se basa en aproximar la derivada por una diferencia finita, lo que evita calcular $f'(x)$ expl√≠citamente.

F√≥rmula

$$x_{n+1}=x_{n}-f(x_{n})\frac{x_{n}-x_{n-1}}{f(x_{n})-f(x_{n-1})}$$

Acotaciones

  • Convergencia superlineal si la aproximaci√≥n inicial es cercana a la ra√≠z.
  • √ötil cuando la derivada es dif√≠cil de calcular, aunque es menos estable que Newton-Raphson.
  • Requiere que $f(x_{n}) \ne f(x_{n-1})$ para evitar divisiones por cero.

3.1. C√≥digo en Python

import math

# Funci√≥n a resolver: f(x) = e^(-x) - x
def f(x):
    return math.exp(-x) - x

def secante(x0, x1, tol=1e-6, max_iter=100):
    """
    Aproxima la ra√≠z de f(x) usando el m√©todo de la Secante.
    
    x0, x1: valores iniciales
    tol: tolerancia para convergencia
    max_iter: n√∫mero m√°ximo de iteraciones
    """
    for n in range(1, max_iter+1):
        f0, f1 = f(x0), f(x1)
        
        if abs(f1 - f0) < 1e-12:
            print("Divisi√≥n por cero. No se puede continuar.")
            return None
            
        x2 = x1 - f1*(x1 - x0)/(f1 - f0)
        
        print(f"Iter {n}: x0={x0:.6f}, x1={x1:.6f}, x2={x2:.6f}, f(x2)={f(x2):.6f}")
        
        if abs(x2 - x1) < tol:
            print(f"\nLa ra√≠z aproximada es: {x2:.6f}")
            return x2
            
        x0, x1 = x1, x2
        
    print("No se alcanz√≥ la tolerancia requerida.")
    return x2

4. M√©todo de Punto Fijo

Definici√≥n

Convierte la ecuaci√≥n $f(x)=0$ en la forma $x=g(x)$ y busca un **punto que permanezca invariable** bajo $g(x)$, utilizando iteraciones sucesivas.

F√≥rmula

$$x_{n+1}=g(x_{n})$$

Acotaciones

  • Converge si $|g'(x)| < 1$ en la vecindad de la ra√≠z.
  • No requiere derivadas, pero depende de una buena elecci√≥n de $g(x)$.
  • La convergencia puede ser lenta o no ocurrir si la funci√≥n no es adecuada.

4.1. C√≥digo en Python

# Buscamos la ra√≠z de x = cos(x), as√≠ que g(x) = cos(x)
def g(x):
    return math.cos(x)

def punto_fijo(x0, tol=1e-6, max_iter=100):
    """
    Aproxima la ra√≠z de x = g(x) usando el m√©todo de Punto Fijo.
    
    x0: valor inicial
    tol: tolerancia para convergencia
    max_iter: n√∫mero m√°ximo de iteraciones
    """
    # Se corrigi√≥ el f-string para alinear la cabecera
    print(f"{'Iter':<5}{'x':<12}{'g(x)':<12}{'Error':<12}")
    print("-"*45)
    
    for n in range(1, max_iter+1):
        x1 = g(x0)
        error = abs(x1 - x0)
        
        print(f"{n:<5}{x1:<12.8f}{g(x1):<12.8f}{error:<12.8f}")
        
        if error < tol:
            print(f"\nLa ra√≠z aproximada es: {x1:.8f}")
            return x1
            
        x0 = x1
        
    print("No se alcanz√≥ la tolerancia requerida.")
    return x1

# Valor inicial distinto para mostrar convergencia desde otro punto

5. M√©todo de Regula Falsi (Falsa Posici√≥n)

Definici√≥n

M√©todo que combina la bisecci√≥n con **interpolaci√≥n lineal** para encontrar ra√≠ces de funciones continuas. Asegura que siempre exista un cambio de signo en el intervalo seleccionado.

F√≥rmula

$$x_{n} = a_n - f(a_n)\frac{b_n - a_n}{f(b_n) - f(a_n)}$$

Acotaciones

  • M√°s r√°pido que la bisecci√≥n para funciones que cambian de signo pronunciadamente.
  • Mantiene siempre un intervalo que contiene la ra√≠z.
  • Puede converger lentamente si la funci√≥n es casi lineal en el intervalo.

5.1. C√≥digo en Python

import math

# Funci√≥n a resolver: f(x) = x^3 - 2x - 5
def f(x):
    return x**3 - 2*x - 5

def regula_falsi(a, b, tol=1e-6, max_iter=100):
    """
    Encuentra la ra√≠z de f(x) en [a,b] usando el m√©todo Regula Falsi.
    
    a, b: extremos del intervalo
    tol: tolerancia para convergencia
    max_iter: n√∫mero m√°ximo de iteraciones
    """
    if f(a)*f(b) >= 0:
        print("No se cumple el teorema del valor intermedio.")
        return None

    # Se corrigi√≥ el f-string para alinear la cabecera
    print(f"{'Iter':<5}{'a':<12}{'b':<12}{'x':<12}{'f(x)':<12}")
    print("-"*55)
    
    for n in range(1, max_iter+1):
        # F√≥rmula de Regula Falsi
        x = b - f(b)*(a - b)/(f(a) - f(b))
        fx = f(x)
        
        print(f"{n:<5}{a:<12.6f}{b:<12.6f}{x:<12.6f}{fx:<12.6f}")
        
        if abs(fx) < tol:
            print(f"\nLa ra√≠z aproximada es: {x:.6f}")
            return x
            
        if f(a)*fx < 0:
            b = x
        else:
            a = x
            
    print("No se alcanz√≥ la tolerancia requerida.")
    return x

# Buscamos la ra√≠z de f(x)=x^3-2x-5 en el intervalo [2, 3]
a, b = 2, 3
